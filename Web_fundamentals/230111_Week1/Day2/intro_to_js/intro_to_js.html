An Introduction to Javascript

Let's see what's so special about JavaScript, what we can achieve with it, and what other technologies play well with it.

What is JavaScript?

JavaScript was initially created to "make web pages alive"

The programs in this language are called scripts. They can be written right in a web page's HTML and run automatically as page loads.

Scripts are provided and executed as plain text. They don't need special preparation or compilation to run.

In this aspect, JavaScript is very different in another language called Java.

When JavaScript was created, It inially had another name: "LiveScript". But Java was very popular at that time, so it was decided that positioning a new language as a "young brother" of Java would help.

But as it evolved, JavaScript became a fully independent language with its own specification calledECMAScript,and now it has no relation to Java at all.

Today, JavaScript can excute not only in browser, but also on the server, or actually on any device that has a special program called The JavaScript Engine.

How do engines work?

Engines are complicated. But the basics are easy.

  1. The engine (embedded if it's a browser) reads ("parses") the script.
  2. Then it converts ("compiles") the script to the machine language.
  3. And then the machine code run, pretty fast.

The engine applies to optimized at each step of the process. It even watches the compiled script as it runs. analyzes the data flows through it, and further optimizes the machine code based on that knowledge.
